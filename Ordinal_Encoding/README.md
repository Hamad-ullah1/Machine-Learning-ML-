Encoding Categorical Data (Ordinal Encoding & Label Encoding)

This repository contains practice work of Machine Learning course. The focus of this session is learning how to encode categorical variables using **Ordinal Encoding** and **Label Encoding**, enabling machine learning models to work with non-numeric data.

## 📌 Topics Covered

- Introduction to Categorical Data
- Types of Categorical Variables
  - Nominal Data
  - Ordinal Data
- Why Encoding is Required
- Label Encoding
- Ordinal Encoding
- Using `OrdinalEncoder`
- Using `LabelEncoder`
- Choosing the Appropriate Encoding Technique

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook
