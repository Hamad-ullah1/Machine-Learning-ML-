Fetching Data From an API

This repository contains practice work of Machine Learning. The focus of this session is learning how to fetch data from APIs and load it into Python for data analysis and machine learning.

## 📌 Topics Covered

- Introduction to APIs
- Understanding API requests and responses
- Working with REST APIs
- Fetching data using Python
- Reading JSON responses
- Converting API responses into Pandas DataFrames
- Basic data exploration after fetching data

## 🛠️ Technologies Used

- Python
- Pandas
- Requests
- JSON
- Jupyter Notebook

## 📂 Project Structure

```
│── Fetching_Data_From_API.ipynb
└── README.md
```

## 📖 Learning Outcomes

After completing this project, I learned how to:

- Fetch data from public APIs using Python.
- Send HTTP requests and handle API responses.
- Parse JSON data returned by APIs.
- Convert API data into Pandas DataFrames.
- Prepare API data for analysis and machine learning tasks.

## 🚀 How to Run

1. Clone this repository.
2. Install the required libraries (if needed):

```bash
pip install pandas requests
```

3. Open the Jupyter Notebook.
4. Run all notebook cells.
