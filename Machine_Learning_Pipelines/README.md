Machine Learning Pipelines

This repository contains practice work of Machine Learning. The focus of this session is learning how to build **Machine Learning Pipelines** using Scikit-learn to streamline data preprocessing and model training into a single, reusable workflow.

## 📌 Topics Covered

- Introduction to Machine Learning Pipelines
- Why Pipelines are Important
- Creating Pipelines with Scikit-learn
- Combining Preprocessing and Model Training
- Integrating ColumnTransformer with Pipelines
- Training and Evaluating Pipeline Models
- Making Predictions with Pipelines
- Benefits of Reusable Machine Learning Workflows

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## 📂 Project Structure

```
│── Machine_Learning_Pipelines.ipynb
└── README.md
```

## 📖 Learning Outcomes

After completing this project, I learned how to:

- Build end-to-end machine learning pipelines using Scikit-learn.
- Combine preprocessing steps and model training into a single workflow.
- Integrate `ColumnTransformer` with `Pipeline`.
- Prevent data leakage by applying transformations correctly.
- Simplify model training, evaluation, and prediction.
- Create reusable and maintainable machine learning workflows.

## 🚀 How to Run

1. Clone this repository.
2. Install the required libraries (if needed):

```bash
pip install pandas numpy scikit-learn
```

3. Open the Jupyter Notebook.
4. Run all notebook cells.
