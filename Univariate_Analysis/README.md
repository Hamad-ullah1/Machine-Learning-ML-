EDA using Univariate Analysis

This repository contains practice work from of Machine Learning. The focus of this session is performing **Exploratory Data Analysis (EDA)** using **Univariate Analysis** to understand the distribution and characteristics of individual features in a dataset.

## 📌 Topics Covered

- Introduction to Exploratory Data Analysis (EDA)
- What is Univariate Analysis?
- Analyzing Numerical Features
- Analyzing Categorical Features
- Measures of Central Tendency
- Measures of Dispersion
- Distribution of Data
- Detecting Outliers
- Interpreting Feature Distributions

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

```
│── Univariate_Analysis.ipynb
```

## 📖 Learning Outcomes

After completing this project, I learned how to:

- Perform univariate analysis on numerical and categorical features.
- Summarize data using descriptive statistics.
- Visualize feature distributions with appropriate plots.
- Identify skewness and potential outliers.
- Gain insights from individual variables before building machine learning models.
