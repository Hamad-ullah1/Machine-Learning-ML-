Function Transformer

This repository contains my practice work of Machine Learning. The focus of this session is learning how to use **FunctionTransformer** in Scikit-learn to apply custom transformations to features as part of a machine learning preprocessing pipeline.

## 📌 Topics Covered

- Introduction to FunctionTransformer
- Why Custom Transformations are Needed
- Applying Mathematical Functions to Features
- Log Transformation
- Square Root Transformation
- Reciprocal Transformation
- Creating Custom Transformation Functions
- Integrating FunctionTransformer with Machine Learning Pipelines

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## 📖 Learning Outcomes

After completing this project, I learned how to:

- Understand the purpose of custom feature transformations.
- Apply mathematical transformations using `FunctionTransformer`.
- Create custom preprocessing functions for machine learning.
- Integrate custom transformations into Scikit-learn pipelines.
- Improve feature distributions using appropriate transformations.
- Build reusable and maintainable preprocessing workflows.
