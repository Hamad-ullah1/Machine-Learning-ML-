Power Transformer (Box-Cox & Yeo-Johnson Transform)

This repository contains practice work of Machine Learning course. The focus of this session is learning how to use **Power Transformations** to make numerical data more normally distributed, reduce skewness, and improve the performance of machine learning models.

## 📌 Topics Covered

- Introduction to Power Transformation
- Why Power Transformation is Important
- Handling Skewed Data
- Box-Cox Transformation
- Yeo-Johnson Transformation
- Using `PowerTransformer` in Scikit-learn
- Comparing Data Before and After Transformation
- Choosing the Appropriate Transformation

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## 📂 Project Structure

```
│── Power_Transformer.ipynb
```

## 📖 Learning Outcomes

After completing this project, I learned how to:

- Understand the purpose of power transformations.
- Reduce skewness in numerical features.
- Apply the Box-Cox transformation to positive-valued data.
- Apply the Yeo-Johnson transformation to data containing zero or negative values.
- Use Scikit-learn's `PowerTransformer` for feature preprocessing.
- Compare feature distributions before and after transformation.
