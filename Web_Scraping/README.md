Fetching Data Using Web Scraping

This repository contains practice work of Machine Learning. The focus of this session is learning how to extract data from websites using **Web Scraping** and convert it into a structured format for data analysis and machine learning.

## 📌 Topics Covered

- Introduction to Web Scraping
- Understanding HTML structure
- Sending HTTP requests
- Parsing HTML content
- Extracting data from web pages
- Creating Pandas DataFrames from scraped data
- Saving scraped data for further analysis

## 🛠️ Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas
- Jupyter Notebook

## 📂 Project Structure

```
│── Web_Scraping.ipynb
└── README.md
```

## 📖 Learning Outcomes

After completing this project, I learned how to:

- Send HTTP requests to websites using Python.
- Parse HTML pages using BeautifulSoup.
- Extract useful information from web pages.
- Store scraped data in Pandas DataFrames.
- Prepare web data for analysis and machine learning tasks.

## 🚀 How to Run

1. Clone this repository.
2. Install the required libraries (if needed):

```bash
pip install requests beautifulsoup4 pandas
```

3. Open the Jupyter Notebook.
4. Run all notebook cells.
